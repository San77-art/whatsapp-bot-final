🚀 BOT WHATSAPP IA - INSTRUÇÕES RÁPIDAS

1️⃣ EDITAR .env
   Abra o arquivo .env
   Troque: COLE_SUA_CHAVE_AQUI
   Por sua chave de: https://console.anthropic.com/
   
   Exemplo: CLAUDE_API_KEY=sk-ant-abc123def456...

2️⃣ INSTALAR
   Abra PowerShell nesta pasta
   Copie: pip install -r requirements.txt
   Aperte ENTER

3️⃣ RODAR BOT (Terminal 1)
   Copie: python bot.py
   Aperte ENTER
   Deve aparecer: "🤖 Bot IA com Claude iniciando..."

4️⃣ TESTAR (Terminal 2 novo)
   Abra outro PowerShell nesta pasta
   Copie: python test.py
   Aperte ENTER
   Deve aparecer respostas do bot

✅ PRONTO! Bot funcionando!

📁 ARQUIVOS:
   .env              → Sua API Key
   requirements.txt  → Bibliotecas
   bot.py            → Bot principal
   test.py           → Teste

🆘 ERRO?
   "API key inválida" → Verifica .env
   "ModuleNotFoundError" → Rode: pip install -r requirements.txt
   "Port 5000 in use" → Mude port=5001 em bot.py

💪 Boa sorte!
